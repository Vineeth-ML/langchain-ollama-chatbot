# LangChain Ollama Chatbot

A simple AI chatbot built with LangChain, Ollama, and Streamlit.

## Project Idea
The idea behind this project is to build a locally running AI chatbot 
that uses Ollama to run open-source LLMs like Gemma, without relying 
on paid APIs. By integrating LangChain for prompt management and 
Streamlit for the UI, this project demonstrates how to build a 
simple yet powerful conversational AI app. LangSmith is used to 
track and monitor the chatbot's responses for better observability.

## Tech Stack
- LangChain
- Ollama (gemma:2b)
- Streamlit
- LangSmith (for tracing)

## Requirements
Make sure you have the following packages installed:

- ipykernel
- python-dotenv
- langchain
- langchain_community
- sentence-transformers
- faiss-cpu
- streamlit

## Features
- Ask any question and get AI-powered responses
- LangSmith tracing enabled
- Simple and clean UI
